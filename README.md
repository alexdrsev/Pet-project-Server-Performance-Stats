## Getting Started

___

1. **Clone the repository**
    
    ```
    git clone https://github.com/alexdrsev/Pet-project-Server-Performance-Stats.git
    cd pet_project_server-stats
    ```
    
2. **Make the script executable**
    
    ```
    chmod +x server-stats.sh
    ```
    
3. **Execute the script**
    
    ```
    ./server-stats.sh
	```
